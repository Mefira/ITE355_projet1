# Calculatrice en C

## Description

Ce projet est une calculatrice interactive en ligne de commande écrite en langage C. Elle permet à l'utilisateur d'effectuer diverses opérations mathématiques, telles que l'addition, la soustraction, la multiplication, la division, et d'autres fonctions avancées comme le calcul de la factorielle et la puissance.

## Fonctionnalités

- Addition de deux nombres
- Soustraction de deux nombres
- Multiplication de deux nombres
- Division de deux nombres
- Division entière de deux entiers
- Calcul du reste modulo
- Calcul de la factorielle d'un entier
- Calcul de la puissance d'un nombre
